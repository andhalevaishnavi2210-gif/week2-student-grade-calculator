# Student Grade Calculator

## Project Description

This project is developed using Python.

The system allows users to enter multiple students and their marks for three subjects.

It calculates:

- Average Marks
- Grade
- Personalized Comment
- Class Statistics

---

## Features

✔ Multiple Student Support

✔ Grade Calculation

✔ Personalized Comments

✔ Input Validation

✔ Error Handling

✔ Statistics Calculation

✔ Formatted Result Table

## Grading System

| Marks | Grade |
| ------- | ------- |
| 90-100 | A |
| 80-89 | B |
| 70-79 | C |
| 60-69 | D |
| 0-59 | F |
